# DIARTIS Portfolio

Premium Next.js portfolio website for DIARTIS, a multidisciplinary designer specializing in graphic design, motion design, and video editing.

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
npm run start
```

## Notes

- Built with Next.js App Router, TypeScript, Tailwind CSS, Framer Motion, Lenis, and Lucide Icons.
- Work is organized by discipline categories, not invented project/client names.
- No portfolio PDF was attached in the workspace, so the current cards use original abstract category visuals that can be replaced with real portfolio exports later.
- SEO includes metadata, Open Graph image generation, `robots.txt`, `sitemap.xml`, and JSON-LD structured data.
